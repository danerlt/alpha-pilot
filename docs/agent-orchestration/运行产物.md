# Run Artifacts 设计

## 目标

将长期状态从模型上下文中移出，避免上下文爆炸。

## 推荐目录

```text
alpha-pilot/runs/<run-id>/
  task.md
  state.json
  plan/
    requirements.md
    design.md
    scope.json
  implement/
    summary.md
    changed-files.json
  ops/
    test-plan.md
    test-report.md
    review-report.md
    pr.md
    merge.md
    deploy.md
  handoff/
    planner-to-developer.yaml
    developer-to-ops.yaml
    ops-to-alpha-pilot.yaml
  public/
    stage-status.md
    final-summary.md
```

## 文件职责

### task.md
原始任务说明，由 alpha-pilot 生成。

### state.json
当前 run 的状态机：
- workflow
- currentStage
- status
- blockedReason
- approvalNeeded

### plan/
由 planner 维护。

### implement/
由 developer 维护。

### ops/
由 ops 内部子阶段维护。

### handoff/
跨角色交接摘要。

### public/
群里可见的阶段性汇报内容。
