# 阶段播报规范

## 必播报阶段

### alpha-pilot
- 新任务接入
- planner -> developer 切换
- developer -> ops 切换
- 阻塞升级
- 最终总结

### planner
- 规划开始
- 规划完成
- 需要决策时

### developer
- 实现开始
- 实现完成
- 收到打回并重新处理时

### ops
- 开始验证与交付
- 验证未通过
- 验证通过并交付完成
- 需要审批时

## 不建议播报
- 每次小修小改
- 原始测试日志
- 原始 review 细项
- 每次 retry 的内部细节
- 逐文件变更流水账
