# 角色输入白名单

## alpha-pilot
允许读取：
- task.md
- state.json
- handoff/*
- public/*
- 用户最新明确指令

## planner
允许读取：
- task.md
- 用户新增约束
- 相关 docs/ 参考材料（按需）

不应读取：
- 全量群聊天历史
- developer / ops 的完整私有会话

## developer
允许读取：
- task.md
- plan/requirements.md
- plan/design.md
- plan/scope.json
- planner-to-developer handoff
- ops 退回的 issue 摘要

不应读取：
- 群里全部历史播报
- ops 的完整测试原始日志

## ops
允许读取：
- task.md
- plan/*
- implement/summary.md
- implement/changed-files.json
- developer-to-ops handoff
- repo/branch/PR 信息

不应读取：
- planner / developer 的完整长会话
- 群里所有阶段播报历史
