# Stage 1 完成 — 基础设施层（2026-05-01）

## 做了什么

按 `docs/superpowers/specs/2026-04-30-fastapi-template-refactor-design.md` v3.6 §6 阶段 1 范围搭建：

### 通用工具 src/utils/

- `log.py` — `init_logger`（幂等）+ `ContextFilter`（自动注入 request_id 到 LogRecord）
- `request_id.py` — `get_request_id` / `current_request_id`（HTTP 链路 ID 读取）
- `uuid.py` — `get_uuid_without_hyphen`（32 字符 hex UUID4）
- `time.py` — `TimeUtils`（北京时间 UTC+8 naive）
- `redis.py` — `get_redis_client` / `get_async_redis`（同步/异步双客户端工厂）
- `json.py` — `AppJSONEncoder`（datetime/Decimal/Enum/Pydantic/dataclass）

### 配置 src/configs/

- `app_configs.py` 重写：8 个子配置类多继承（ServiceConfig / CORSConfig / PostgreSQLConfig / RedisConfig / SchedulerConfig / ExchangeConfig / LLMConfig / RiskConfig / SecurityConfig）
- PostgreSQLConfig.db_uri：DATABASE_URL postgres:// 时优先用，否则 PG_* 拼接
- SchedulerConfig 含 TASK_QUEUE_KEY / EVENT_BUS_CHANNEL / SCHEDULER_GRACEFUL_SHUTDOWN_SECONDS 等

### DB 层 src/db/

- `engines.py` — get_engine / get_session_factory 单例（PostgreSQL 同步 + sqlite 兼容占位）
- `session.py` — get_db / get_db_session / CurrentSession（FastAPI Depends + with 语句）
- `src/shared/db.py` 改为转发 wrapper

### 通用层 src/common/

- `response/` — Response[T] + ErrorCode 枚举（含业务码 6xxxxx）+ response_base 便捷构造器
- `exception/errors.py` — AppBaseException 树（自动 ERROR + 调用栈 + request_id）
- `exception/exception_handler.py` — 全局 handler（业务异常 HTTP 200 + body success:false；未识别 Exception 兜底 + traceback；HTTPException 走 FastAPI 默认）
- `api_response.py` — @api_response 装饰器（占位备用）
- `schemas/pagination.py` — Paginated[T] 泛型（items/total/page_index/page_size/pages）
- `events.py` — BaseEvent 占位

### 中间件 src/middleware/

- `request_logging_middleware.py` — req/resp 时间 + 路径 access log
- `error_logging_middleware.py` — 未捕获异常兜底

### 应用工厂 src/app/

- `__init__.py` 提级（迁原 src/app/app.py 内容）→ `from src.app import app`
- 注册中间件栈（CorrelationId 最外层 + RequestLogging + ErrorLogging）
- CorrelationIdMiddleware 用 `get_uuid_without_hyphen` generator → request_id 32 字符 hex
- 注册全局 exception handler（阶段 4 切 Response[T] 时自动生效）
- `src/app/app.py` 改为 re-export shim（兼容 `from src.app.app import app` 的 10 个测试文件）

### 测试基础设施

- `tests/conftest.py` 增加 fixture：
  - `_silence_app_exception_autolog` 全局静音 auto_log
  - `_reset_logger_disabled_state` 重置 logger.disabled / propagate（应对 testcontainers 副作用）
- 新增 `tests/unit/utils/`、`tests/unit/common/`、`tests/integration/` 三组测试目录
