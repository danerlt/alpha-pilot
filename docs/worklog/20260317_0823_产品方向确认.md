# 2026-03-17 Product Direction Adopted

## 决策

参考外部项目后，AlphaPilot 采纳以下产品方向：

1. AI Trader / Program Trader 双模式
2. 多智能体职责分工
3. 交易归因分析（Attribution）
4. 因子智能 / 因子库方向
5. Web 控制台 + 消息端提醒/诊断协同

## 明确不纳入近期范围

- 多交易所支持
- 多模型同台 PK / 竞技场模式

## 已落地到文档

- `docs/产品需求文档.md`
- `docs/认证与管理后台规格说明.md`
- `docs/认证与管理后台计划.md`

## 影响

后续 AlphaPilot 的产品路线将不只围绕“单个 AI 交易闭环”，而是逐步演进为：
- 执行核心（Execution Core）
- 管理与控制平面（Control Plane）
- 策略智能层（Strategy Intelligence）
- 因子研究层（Factor Intelligence）
