# 2026-03-17 Mobile Dashboard Cards

## 做了什么

进一步优化 dashboard 的移动端体验：

1. 将以下模块在手机端改为卡片列表视图：
   - 当前持仓
   - 风控事件
   - 交易记录
2. 桌面端仍保留表格视图
3. 手机端隐藏横向滚动表格，改为更适合单手浏览的卡片布局
4. 保留关键操作：
   - 持仓卡片中可直接平仓
   - 风控事件卡片中可直接解除熔断
   - 交易记录卡片中保留盈亏、时长、市场状态等核心信息

## 为什么做

此前这些区域在手机端主要依赖横向滚动表格，可读性和操作性都不理想。改成卡片后：

- 信息层级更清楚
- 减少横向滚动
- 更适合小屏快速浏览和点击操作

## 如何验证

- `cd frontend && npm run build`
- `cd backend && .venv/bin/pytest -q`

结果：
- 前端 build 通过
