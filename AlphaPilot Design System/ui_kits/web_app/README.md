# AlphaPilot Web UI Kit

PC Web 全功能控制台。**7 个页面** + **3 种 AI 决策卡变体**（Tweaks 切换）。

## 页面
| 页面 | 说明 |
|-----|------|
| 主控制台 | AI 决策卡（hero）+ 权益曲线 + 关键指标 + 持仓预览 + 事件流（粘性右栏） |
| AI 决策流 | 决策列表，按"已执行/已拦截"过滤；使用当前 variant 渲染 |
| 持仓与订单 | 持仓详情表（SL/TP/策略/操作）+ 订单簿 |
| 回测与绩效 | 策略 vs HODL 对比曲线 + 月度 PnL 直方图 + 统计表 |
| 策略与风控 | 受限策略集（可启停）+ 硬风控阈值（只读说明）+ 交易对管理 |
| 审计日志 | 事件流 + AI 日报 |
| 设置 | 占位 |

## AI 卡片 3 种变体
- **Stepper** — 水平流水线（市场快照 → AI → 守卫 → 裁决 → 执行），参数区 5 栏
- **Timeline** — 垂直时间线，每步带图标 + 时间戳
- **Graph** — 节点图（features → LLM → guards → output），显式数据流

## Shell
- 左侧栏（240px）：品牌 / 权益 / 主导航（带活跃指示条）/ 引擎状态
- 顶栏：页面标题 + 风控状态胶囊 + ⌘K 搜索 + 通知 + AUTO 开关

## Tweaks
- `variant` — AI 卡片样式
- `scene` — profit / warn / halted（影响顶栏状态胶囊）
